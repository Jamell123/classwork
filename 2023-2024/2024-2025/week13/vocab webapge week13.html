uniformn Resource loxator (URL)

a text string that specifies the location of a resource on the internet, such as a webpage, image, or video.

firewall

part of a computer system or network which is designed to block unauthorized access while permitting outward communication.

Transmission Control Protocol-Internet Protocol (TCP-IP)

a suite of communication protocols that defines how data is transmitted across networks, particularly the internet, ensuring reliable and ordered delivery of data packets

Network Protocols

established sets of rules that define how data is transmitted and managed between devices on a network, enabling communication

Computer Network

a system of interconnected devices that share data and resources, enabling communication and collaboration through standardized protocols and physical or wireless technologies.

IPv4

the fourth version of the Internet Protocol, a set of rules that govern how data is transmitted over the internet, using a 32-bit address system.

ipv6

the latest version of the Internet protocol, designed to address the issue of IPv4 address depletion by providing a much larger address space, enabling more devices to connect to the internet.

Domain Name Services (DNS)

the internet's directory, translating human-readable domain names (like "google.com") into the numerical IP addresses (like "192.0.2.1") that computers use to communicate.

Hyper-text transfer protocol (HTTP/HTTPS)

the foundation of data exchange on the web, allowing browsers and servers to communicate, while HTTPS (Hypertext Transfer Protocol Secure) is the secure version of HTTP, encrypting data transfer for privacy and security.

cloud computing

he on-demand delivery of computing resources, like servers, storage, databases, networking, and software, over the internet, allowing users to access and use these resources without managing the underlying infrastructure

OSI Model

a conceptual framework that describes how network communication functions are organized into seven distinct layers, providing a standardized way to understand and troubleshoot network interactions.

Virtual Private network (VPN)

a secure, encrypted connection that allows a device to connect to a network over the internet, masking your IP address and encrypting your internet traffic for enhanced privacy and security.
